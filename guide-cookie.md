# OneLauncher 로그인을 위한 쿠키 가져오는 방법 설명서
본 문서는 OneLauncher에서 쿠키를 직접 입력하여 로그인하는 방법을 사용할 때, 기존에 사용하던 브라우저에서 로그인 한 쿠키를 가져오는 방법을 설명합니다.

## 1. 기존 브라우저로 로그인
브라우저의 로그인 상태를 OneLauncher로 옮기기 위해서는 기존에 사용하던 브라우저에서 로그인을 해야합니다. \
로그인 절차를 마친 후, 2. 으로 넘어갑니다.

## 2. 쿠키 가져오기
브라우저별로 약간의 차이는 있지만, 크게 다르지 않은 과정을 거칩니다. \
`확장 기능` 또는 `확장 프로그램` 등을 통해, [라스트오리진 홈페이지](https://last-origin.valofe.com/) 또는 [밸로프 홈페이지](https://valofe.com)의 쿠키를 추출합니다. \
추출된 쿠키는 `A=B;C=D;...` 형태여야 합니다.

## 3. 입력하기
추출된 쿠키를 OneLauncher의 쿠키 입력 화면에 입력합니다. \
입력된 쿠키가 유요하면 로그인에 성공한 화면으로 전환됩니다.
